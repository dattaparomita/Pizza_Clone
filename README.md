🍕 Pizza Pie Landing Page: 
 A modern, responsive landing page for a pizza restaurant called Pizza Pie. This project features a clean UI with a focus on high-quality food photography and clear call-to-action buttons.

# Key Features:
 Responsive Navigation: A clean header with links for Home, About, Menu, Service, and Contact that adapts to different screen sizes.
 Hero Section: A minimalist left-aligned hero section with a bold "Pizza Taste" headline and a compelling sub-headline.
 Dark Mode Toggle: An integrated moon icon in the navigation bar, ready for implementing light/dark mode functionality.
 High-Resolution Visuals: A large, appetizing hero image of a pizza on a wooden board to grab user attention immediately.

 CSS
